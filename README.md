# Spam Message Detector

A Python-based SMS spam detection system that classifies messages as **spam**, **not spam**, or **potential spam** using machine learning models. The system uses Naive Bayes and Logistic Regression algorithms and is trained with a dataset containing both regular and tricky spam messages.

## Features

- Detects spam messages in SMS
- Shows spam probability for each message
- Handles tricky messages containing URLs, offers, and urgent alerts
- Uses Naive Bayes and Logistic Regression models for prediction
